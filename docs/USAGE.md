# Usage Guide

How to use the AI Microlearning LMS application.

## User Roles

### Learner

Learners interact with the AI tutor through multimodal conversation (text or voice) to learn from atomic learning nuggets.

### Admin

Admins manage content ingestion, view nuggets, configure settings, and monitor system analytics.

## Learner Features

### Starting a Learning Session

1. Navigate to the learner dashboard
2. Click "Start Learning Session"
3. Choose mode: Text or Voice
4. Begin conversation with AI tutor

### Learning Interface

- **Chat Interface:** Text-based conversation with AI tutor
- **Media Widget:** Displays slides, images, and audio for learning nuggets
- **Progress Panel:** Shows mastery scores and knowledge gaps
- **Narrative Tree:** Visualizes learning path and available choices

### Voice Mode

- Click voice mode button to switch to 2-way voice conversation
- Speak naturally - AI responds with voice
- Switch back to text mode anytime

## Admin Features

### Content Ingestion

#### File Watching

1. Navigate to Ingestion Management
2. Click "Add Watched Folder"
3. Specify folder path
4. Files in folder are automatically processed

#### URL Monitoring

1. Navigate to Ingestion Management
2. Click "Add Monitored URL"
3. Enter URL to monitor
4. System checks for changes periodically

### Nugget Management

1. Navigate to Nugget Store
2. Browse or search nuggets
3. Click nugget to view details
4. Edit content if needed
5. Regenerate slides/audio if desired

### Settings

1. Navigate to Settings
2. Configure system settings
3. Update API keys
4. Adjust storage paths
5. Save changes

### Analytics

1. Navigate to Analytics Dashboard
2. View usage metrics
3. Monitor costs
4. Track learner engagement

## Best Practices

### For Learners

- Engage naturally in conversation
- Ask questions when unclear
- Review progress regularly
- Explore different narrative paths

### For Admins

- Monitor ingestion jobs regularly
- Review generated nuggets for quality
- Optimize API usage for cost control
- Keep documentation updated
